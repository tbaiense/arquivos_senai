Sobre variáveis:

  • Podem ser declaradas com 
    var
    e 
    let
  • Sofrem o efeito hoisting, que move as declarações das variáveis 
    let
    para o início do escopo
  • Podem sofrer ghosting, quando são declaradas dentro de um program block usando 
    let
    .
    Ou seja, redeclarações de variáveis 
    let
    em escopos aninhados criam uma variável local, mantendo a outra intacta no processo.

Sobre arrays:

  • Podem ser passadas como argumentos para funções
  • Não precisam ter seu tamanho especificado durante a declaração
  • São considerados instâncias do tipo 
    Object
    , portanto possuem propriedades e keys próprias
